# SQL Queries Performed
- The following SQL concepts were covered as required in the task:

# Basic Queries
- SELECT * to understand data structure
- DISTINCT to identify unique categories

# Filtering & Sorting
- WHERE clause for filtering (e.g., Technology category)
- ORDER BY for sorting sales in descending order

# Aggregations & Grouping
- SUM(sales) for total sales
- AVG(profit) for average profit
- COUNT(*) for total orders
- GROUP BY category, region, and segment

# HAVING Clause
- Filtered aggregated results (e.g., categories with total sales > 100000)

# Advanced Conditions
- BETWEEN for date range filtering
- LIKE for customer name pattern search

# Business Query
- Identified Top 5 customers by total sales
